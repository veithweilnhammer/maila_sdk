MAILA Sandbox

MAILA infers latent mental states from everyday interaction patterns. This page shows a minimal end-to-end flow: record interaction data, send it to the backend API, and request predictions.

How to:

Set the backend URL. For local piloting, use http://localhost:8000.

Start recording sends event batches to POST /exp_main.

Predict flushes pending data and calls POST /predict.

Predict from uploaded file sends a saved recording to POST /predict-upload.

Notes:
  • Short runs are fine for testing connectivity and JSON responses.
  • Use longer interaction windows for more stable predictions.
  • Accepted upload formats: .jsonl, .json, .txt.
Example recording file: example.jsonl
